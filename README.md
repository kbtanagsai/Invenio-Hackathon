# Smart Parking Reservation System

## Overview

This project was developed during the Invenio Hackathon as an attempt to create a simple Smart Parking Reservation System. The goal was to provide users with a visual interface for reserving and managing parking spaces efficiently.

As first-year engineering students with limited programming experience, our primary objective was to learn, experiment, and build a working prototype within the hackathon timeframe. While the project was not fully completed, it helped us gain practical exposure to Python programming, GUI development, databases, and collaborative problem-solving.

## Features Implemented

* Interactive parking lot grid interface using Python Tkinter.
* Parking spot reservation functionality.
* Reservation cancellation functionality.
* Storage of reservation data using SQLite database.
* Real-time status updates showing available and occupied parking spots.

## Technologies Used

* Python
* Tkinter (GUI)
* SQLite Database

## Current Limitations

The project remains a prototype and several planned features were not completed due to time constraints and our limited development experience at the time. Some intended future improvements include:

* User authentication and login system.
* Vehicle information management.
* Advanced parking allocation algorithms.
* Cloud database integration.
* Mobile/Web application support.
* Real-time parking analytics and monitoring.

## Learning Outcomes

This hackathon project provided valuable experience in:

* Python programming fundamentals.
* Database integration using SQLite.
* GUI design with Tkinter.
* Team collaboration and project management.
* Rapid prototyping under time constraints.
